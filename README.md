# Google Maps Platform Demo

A demo for the Google Maps Platform.
You can see a final, live version of this code [here](https://gmap-workshop.web.app/).

## Instructions

First

```
npm install
```

then

```
gulp build-serve
```

and

```
gulp watch
```
